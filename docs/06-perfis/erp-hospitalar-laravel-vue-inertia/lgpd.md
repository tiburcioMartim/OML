# LGPD — ERP Hospitalar

## Dados sensíveis
- Dados de saúde (prontuário, exames, prescrições)
- Dados pessoais (CPF, endereço, telefone)
- Dados financeiros (cobranças, convênios)

## Regra
Não presumir que dados de saúde podem ser excluídos. Avaliar obrigação legal, regulatória, assistencial e contábil.
