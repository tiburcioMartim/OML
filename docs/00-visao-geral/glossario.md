# Glossário do OML

## Objetivo

Definir termos importantes usados pelo OML para garantir consistência e clareza.

## Quando usar

Sempre que houver dúvida sobre o significado de um termo no contexto do OML.

---

## Termos do OML

| Termo | Definição |
|---|---|
| **OML** | Orquestrador de Migração Legada |
| **Legado** | Sistema antigo que será modernizado |
| **Microfase** | Unidade mínima de trabalho do OML |
| **Fase** | Conjunto de microfases com objetivo maior |
| **Guardião** | Conjunto de regras e checklists para um domínio técnico |
| **Dossiê** | Documento detalhado que descreve uma tela |
| **Paridade** | Comparação entre comportamento do legado e do novo |
| **Homologação** | Validação pelo usuário final |
| **Rollback** | Voltar para a versão anterior |
| **Rollout** | Implantação gradual e controlada |
| **Go-live** | Momento em que o novo sistema entra em produção |
| **Modo sombra** | Novo sistema roda em paralelo sem ser oficial |
| **A INVESTIGAR** | Marcação para informação não confirmada |
| **HIPÓTESE NÃO VALIDADA** | Marcação para suposição sem evidência |

---

## Termos técnicos

| Termo | Definição |
|---|---|
| **RPO** | Recovery Point Objective — quanto de perda de dados é aceitável |
| **RTO** | Recovery Time Objective — quanto tempo de indisponibilidade é aceitável |
| **Failover** | Troca para ambiente alternativo em caso de falha |
| **Idempotência** | Garantia de que repetir uma operação não causa duplicidade |
| **Circuit breaker** | Padrão para evitar chamadas a serviço indisponível |
| **Feature flag** | Chave para ativar/desativar funcionalidade sem deploy |
| **RBAC** | Controle de acesso baseado em papéis |
| **LGPD** | Lei Geral de Proteção de Dados |
| **WCAG** | Web Content Accessibility Guidelines |
| **OWASP ASVS** | Application Security Verification Standard |

---

## Termos do domínio hospitalar

| Termo | Definição |
|---|---|
| **Prontuário** | Registro médico do paciente |
| **Convênio** | Plano de saúde ou seguradora |
| **Guia** | Documento de autorização de procedimento |
| **Triagem** | Avaliação inicial do paciente |
| **Faturamento** | Processo de cobrança de serviços |

> Este glossário deve ser expandido conforme novos termos surgirem durante a migração.
