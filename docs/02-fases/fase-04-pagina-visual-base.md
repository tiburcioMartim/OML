# Fase 04 — Página Visual Base

## Objetivo
Criar a Página Visual base com tokens, componentes e IDs únicos antes da primeira tela migrada.

## Ações obrigatórias
- [ ] Conduzir entrevista de design (30 perguntas)
- [ ] Definir cores, tipografia, espaçamentos, bordas, sombras
- [ ] Criar componentes base (botões, inputs, tabelas, modais, etc.)
- [ ] Atribuir IDs únicos a todos os elementos
- [ ] Documentar dark mode, light mode, alto contraste
- [ ] Documentar responsividade (breakpoints)
- [ ] Documentar estados (loading, vazio, erro, sucesso)
