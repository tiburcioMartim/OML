# /migracao-status

## Objetivo

Mostrar o status geral da migração: telas no backlog, telas em andamento, telas concluídas, pendências e riscos.

## Quando usar

A qualquer momento para ter visão geral do progresso.

## Entradas esperadas

Nenhuma. Lê os arquivos de memória.

## Ações obrigatórias

- [ ] Ler `memoria/backlog-telas.yaml`
- [ ] Contar telas por status
- [ ] Listar telas em andamento
- [ ] Listar pendências de `memoria/pendencias.md`
- [ ] Listar riscos de `memoria/riscos.md`
- [ ] Mostrar próxima tela recomendada

## Ações proibidas

- [ ] Não alterar nenhum arquivo
- [ ] Não implementar nada

## Arquivos que atualiza

Nenhum. Apenas leitura.

## Frase obrigatória de encerramento

"Status atualizado. [N] telas no backlog, [N] em andamento, [N] concluídas."

## Próximo passo

Depende do estado atual. O OML sugere o comando mais adequado.
